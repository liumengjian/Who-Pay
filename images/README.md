# 图片资源说明

请从 [iconfont项目](https://www.iconfont.cn/manage/index?manage_type=myprojects&projectId=5085209) 下载合适的图标。

## 下载步骤

1. 访问 iconfont 项目页面
2. 选择合适的图标，建议下载以下图标：
   - **首页/活动图标**：选择"首页"或"活动"相关的图标
   - **用户/我的图标**：选择"用户"或"个人中心"相关的图标
   - **Logo图标**：可以选择"钱包"、"AA"、"账单"等相关图标

3. 下载设置：
   - 格式：PNG
   - 尺寸：选择 81x81px（Tab图标）或 160x160px（Logo）
   - 颜色：未选中状态使用灰色（#7A7E83），选中状态使用主题色（#07c160）

## 必需图片文件

请将下载的图标重命名为以下文件名并放置在此目录：

1. **logo.png**
   - 尺寸：160x160px
   - 用途：登录页Logo
   - 建议：使用品牌色或渐变色

2. **default-avatar.png**
   - 尺寸：120x120px
   - 用途：默认用户头像
   - 建议：使用默认头像图标

3. **home.png** / **home-active.png**
   - 尺寸：81x81px
   - 用途：活动Tab图标（未选中/选中状态）
   - 颜色：home.png使用灰色，home-active.png使用主题色（#07c160）

4. **user.png** / **user-active.png**
   - 尺寸：81x81px
   - 用途：我的Tab图标（未选中/选中状态）
   - 颜色：user.png使用灰色，user-active.png使用主题色（#07c160）

## 快速生成方法

如果iconfont没有提供PNG格式，可以：
1. 下载SVG格式
2. 使用在线工具转换为PNG（如 https://convertio.co/zh/svg-png/）
3. 调整尺寸和颜色后保存
